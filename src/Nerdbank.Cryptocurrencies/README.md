# Nerdbank.Cryptocurrencies

## Features

* Cryptography functions that are common to many cryptocurrencies:
  * Base58Check
  * Bech32 and Bech32m
  * Blake2B
  * Compact Size

BIPs:
  * BIP-32 Hierarchical Deterministic wallets
  * BIP-39 seed phrases
  * BIP-44 Multi-Account Hierarchy for Deterministic Wallets
